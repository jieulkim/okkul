1. 코딩 테스트 대비 알고리즘 문제 풀이
- 다양한 알고리즘 유형의 문제를 풀며 코딩 테스트를 대비함
- 문제 해결 과정에서 시간 복잡도와 자료구조 선택 기준을 함께 점검
- 풀이 후 로직을 정리하며 접근 방식 개선 및 복습 진행

2. 요구사항 정리
- 각 기능별 동작 흐름과 예외 상황을 고려하여 요구사항 명세서 작성
-사용자 행동을 기준으로 기능 흐름과 필수, 선택 기능을 정의

3. ERD 설계
- 핵심 도메인 기준으로 데이터베이스 구조를 설계
- 엔티티 간 관계를 정의하고, 테이블 구조 및 주요 컬럼 도출
- 추후 확장성을 고려하여 중복 데이터 최소화 방향으로 ERD 구성
