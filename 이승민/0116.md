# ERD Cloud 학습 요약

## 1. 개요
* **정의:** 웹 브라우저에서 실행되는 무료 데이터베이스 모델링(ERD) 도구
* **URL:** [erdcloud.com](https://www.erdcloud.com)
* **핵심:** 별도 설치 없이 DB 설계 및 팀 협업 가능

## 2. 주요 특징
* **Web-Based:** 인터넷만 되면 OS 관계없이 접속 가능
* **실시간 협업:** 팀원 초대 및 동시 편집 지원
* **SQL 자동 생성:** 설계된 모델을 Oracle, MySQL 등 DBMS별 DDL로 추출
* **논리/물리 동시 지원:**
  * **논리:** 한글 명칭 (기획/소통용)
  * **물리:** 영문 컬럼명, 타입 (실제 DB용)

## 3. 장단점 비교

| 구분 | 내용 |
| :--- | :--- |
| **장점** | • **무료 & 무설치** (높은 접근성)<br>• **URL 공유**로 쉬운 협업<br>• **정보공학(IE) 표기법** 준수 |
| **단점** | • 사용자 급증 시 **서버 불안정**<br>• 기업용으로는 아쉬운 **보안성**<br>• 전문 툴 대비 둔탁한 **UI 조작감** |

## 4. 사용 흐름 (Workflow)
1. **설정:** 회원가입 → 새 ERD 생성 (타겟 DBMS 선택)
2. **설계:** 테이블 추가 → 논리/물리 이름 및 컬럼(PK, Type) 정의
3. **관계 설정:**
   * 1:1, 1:N, N:M 관계선 연결
   * 실선(식별) vs 점선(비식별) 구분
4. **내보내기:** 메뉴 > 내보내기 > SQL 복사 > DB 클라이언트 실행

## 5. 경쟁 툴
* **AQueryTool:** 한국형 웹 툴, 직관적이나 무료 제한 있음
* **MySQL Workbench:** 설치형, 안정적이나 웹 협업 불가
