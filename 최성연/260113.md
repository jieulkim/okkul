
## 2026.01.13

### 1. 프로젝트 기획 구체화 회의

주제 선정 : OPIc 시험 준비를 도와주는 AI 튜터 서비스

1) 문제 정의
- 오픽 경험과 관련한 페인 포인트를 분석하기 위해 싸피생 및 오픽시험 경험자 대상으로 설문조사 진행(약 65명)
- 설문조사 토대로 페인포인트 3순위 정리

```
1. **피드백 부재 (내가 말하는 문장, 발음, 억양이 맞는지 확신이 없음)**
    1. 틀린 문법에 대한 피드백
    2. 발음에 대한 피드백
    3. 어휘나 표현에 대한 피드백
    4. 롤플레잉, 사회적 문제 질문에 대해서 논리 검증 피드백
    5. 속도에 대한 피드백
    6. 불필요한 표현에 대한 반복 
2. **소재 부족 (돌발 질문에 대해 내 경험을 즉석에서 떠올리고 영작하기 힘듦)**
    1. 내가 접하지 못한 정보 / 상황별 영작 스크립트
    2. 사용자 표현 위주의 영작
    3. 설문조사, 이전 답변 데이터를 바탕으로 영작
3. 공부 방법 수립의 어려움 (오픽 준비를 어떻게 해야 하는지, 내가 잘 하고 있는 건지 잘 모르겠음)
```

2) 사용자 정의
> 서비스 대상을 더 구체화

```
- 사용자 : 오픽 시험을 앞둔 취준생/대학생
- 상황 : 오픽을 일주일~하루 앞둔 상황
- 불편 : 비용이 부담돼서 유료 서비스를 사용하지 못하고, 오픽 답변에 대한 피드백을 받기 어려움, 선택한 주제에 대해서 나의 답변을 정리하기 어려움
- 목표 : 가격 부담없이 오픽 시험에 익숙해지기, 어떤 토픽이 나오든 내 생각을 영어로 표현하기
```

3) 사용 시나리오
- 2-3개 정도 핵심이라고 생각하는 사용자 플로우를 정의하기
- 코드로 작성하기 전에 글로 쓰는 단계
    
```
- 사용자 회원가입/로그인
- 튜토리얼(나중 우선순위)
- 오픽 시험 기능
    - 할 때마다 오픽 설문조사 실시(이전에 선택한 내용 저장해두기)
    - 모드1) 유형별로 연습
        - 유형 선택
            - 자기소개
            - 3세트 문제
            - 롤플레잉
            - 2세트 문제(사회적 문제)
        - 토픽 선택
        - 선택한 유형과 토픽에 대한 연습 진행
        - 오픽 채점 기준에 따라 사용자의 답변 피드백
            - 동사 시제 / 다양한 형용사 / 적절한 접속사 / 발화량
            - + 소재에 대한 영작 추천
                - 적절한 예시 답변
    - 모드2) 모의고사 (15문제)
        - 설문조사 선택
        - 모의고사 진행
        - 채점 기준에 대한 평가표 제공(선택)
- 피드백 제공
    - 유창성 및 발화 패턴
    - 문법 및 어휘
    - 논리 구조 및 내용
```

(기능 정의 단계부터 내일 수행 예정)
