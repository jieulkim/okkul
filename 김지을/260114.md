1. 코딩 테스트 준비
- 알고리즘 문제 풀이를 통해 코딩 테스트를 대비함
- 시간, 공간 복잡도를 고려한 풀이를 연습하며 코드 개선을 진행함

2. 기능 구체화
- 서비스 전반의 기능을 점검하며 핵심 기능과 세부 기능을 정리함
- 사용자 액션 기준으로 기능 흐름과 예외 케이스를 정의함
- 구현 단계에서 혼선을 줄이기 위해 기능 단위로 정리함

3. 화면 단위, 기능 단위 쪼개기
- 서비스에 필요한 화면을 도출하고 화면별 역할을 정의함
- 각 화면에서 제공되어야 할 기능을 기능 단위로 세분화함
- 화면 간 이동 흐름과 기능 간 연결 관계를 정리함
