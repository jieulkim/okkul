# 🗣️ AI 기반 오픽(OPIc) 튜터 서비스 (Project Name)

"비용 부담 없이, 언제 어디서나 내 답변에 대한 완벽한 피드백을." > 오픽 수험생을 위한 맞춤형 AI 모의고사 및 피드백 서비스입니다.

## 1. 프로젝트 배경 및 문제 정의 (Problem Definition)

우리는 오픽 준비생들이 겪는 다음과 같은 세 가지 핵심 문제를 해결하고자 합니다.

피드백의 부재: 내가 말하는 문장, 발음, 억양이 맞는지 확신이 없으며, 논리 구조나 어휘에 대한 즉각적인 교정이 어려움.

소재의 부족: 돌발 질문에 대처하기 힘들고, 나만의 경험을 영어로 즉석에서 작문하는 것에 대한 어려움.

학습 방향의 모호함: 현재 내 실력이 어느 정도인지, 어떤 커리큘럼으로 공부해야 목표 등급을 받을 수 있는지 알기 어려움.

## 2. 타겟 사용자 (Target Audience)

👤 페르소나

- 대상: 오픽 시험을 앞둔 취준생 및 대학생

- 상황: 시험이 일주일~하루 남은 시점, 단기간 고효율 학습 필요

- Pain Point:

  - 유료 학원/첨삭 서비스 비용이 부담됨.
  - 내가 선택한 주제(Survey)에 맞춰 나만의 답변을 정리하고 싶음.

- Goal: 비용 부담 없이 시험 환경에 익숙해지고, 어떤 돌발 토픽이 나와도 내 생각을 영어로 표현할 수 있는 능력 배양.

## 3. 주요 기능 (Key Features)

### ✅ 공통 기능 (User Management)

- [ ] 회원가입/로그인: 소셜 로그인 및 일반 계정 지원.

- [ ] 마이페이지: 학습 히스토리 관리, 개인화된 답변 스토리(User Story) 저장 및 수정.

- [ ] 튜토리얼: 서비스 최초 접속 시 사용 가이드 제공.

### 📝 모드 1: 유형별 연습 (Practice Mode)

사용자가 부족한 유형을 집중적으로 공략하는 모드입니다.

- 설문조사(Background Survey): 오픽 문항 선택과 동일한 UI 제공 (기존 데이터 불러오기 가능).

- 유형 선택: 자기소개, 묘사/설명, 롤플레잉, 비교/이슈 중 선택.

- 학습 진행:

  - 선택한 주제 + 돌발 주제 조합의 문제 세트 제공 (TTS/Audio).

  - 질문 텍스트 보기 옵션 제공.

  - 타이머(1~2분)와 함께 답변 녹음.

- AI 피드백:

  - STT(Speech-to-Text): 답변 내용 텍스트 변환.

  - 객관적 평가: 문법, 시제, 문장 논리성, 어휘 다양성, 발화량 분석.

  - 비언어적 평가: 자연스러움(Filler words 사용 등), 자신감 분석.

  - 영작 추천: 사용자 답변을 바탕으로 더 나은 표현 및 예시 답안 제안.

### 🏆 모드 2: 실전 모의고사 (Mock Exam Mode)

실제 시험과 동일한 환경에서 15문제를 연속으로 수행하는 모드입니다.

- 프로세스: Survey 선택 → 15문제 연속 출제 (오디오 재생 및 녹음) → 서버 전송.

- 종합 리포트:

  - 예상 등급 (AL, IH, IM 등) 및 총평.

  - 전체 어휘량, 발화량, 주제 적합도 그래프 분석.

  - 문항별 상세 피드백 및 다시 듣기 기능.

- 재피드백(Re-feedback): 답변에 대한 추가 정보를 입력하여 더 정교한 수정 답안 요청 가능.

## 4. 사용자 흐름 (User Flow)

1. 접속 및 로그인: 회원가입 후 메인 대시보드 진입.

2. 모드 선택: '유형별 연습' vs '실전 모의고사' 중 택 1.

3. 설문 조사(Survey): 관심사 및 난이도 설정 (이력 저장 및 불러오기).

4. 문제 풀이: AI가 생성한 질문 청취 → 답변 녹음 → 전송.

5. 분석 및 피드백:

   - STT 변환 텍스트 확인.

   - 문법/어휘/유창성 점수 확인.

   - 개선된 스크립트(모범 답안) 학습.

6. 아카이빙: 나만의 답변 스토리를 '마이페이지'에 저장하여 시험 직전 복습.
